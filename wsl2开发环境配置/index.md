# WSL2中C/C++开发环境配置

## 基础
- 编辑器: LunarVim
- 基于clangd的代码提示, compile_commands.json由bear生成, 命令是`bear -- make`
## 美化
- WSL2: 运行在Windows Terminal中, 安装oh my posh, One Half Dark主题
- 字体: Fira Code Nerd Font (为了显示LunarVim以及终端中各种小图标)
